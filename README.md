Cypher est une application de gestion de musique où les utilisateurs peuvent explorer leur bibliothèque musicale, créer des playlists, et organiser leurs chansons préférées dans des catégories personnalisées. Le projet utilise Flutter pour l’interface utilisateur et l'API Deezer pour accéder aux morceaux et aux informations musicales.

Fonctionnalités
1. Bibliothèque musicale

Les utilisateurs peuvent voir toutes les chansons de leur bibliothèque musicale, avec des informations comme le nom de l'artiste, l'album, et la durée du morceau.

2. Catégories musicales

Les utilisateurs peuvent organiser leurs morceaux dans différentes catégories (par exemple : Rock, Pop, Rap, etc.) pour une navigation plus facile.

3. Playlists personnalisées

Les utilisateurs peuvent créer et gérer des playlists. Chaque playlist peut contenir une sélection de morceaux que l'utilisateur aime ou souhaite écouter en boucle.

Outils utilisé : 
Flutter ,Dart ,API deezer
